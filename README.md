# Todo list app

## cloning project

```
git clone https://github.com/aphrodeosubarno/node-js-todo-app.git
```

## install dependencies

```
yarn install
```

<p>or</p>

```
npm install
```

## run server

```
yarn start
```

<p>Open new terminal</p>

```
npm start
```

## compile SASS/SCSS

<p>Open new terminal</p>

```
yarn run sass
```

<p>Open new terminal</p>

```
npm run sass
```

## open this server at your browser

```
http://localhost:8080
```
